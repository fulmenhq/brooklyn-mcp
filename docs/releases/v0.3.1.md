## Overview

Brooklyn v0.3.1 finalizes the FulmenHQ release signing workflow and updates safe dependencies. This is a maintenance release with no breaking changes.

---

## What's New

### Release Signing Workflow

- **Environment variable standardization**: Changed `RELEASE_TAG` to `BROOKLYN_RELEASE_TAG` throughout Makefile and documentation for consistency with project naming conventions.
- **New Makefile targets**:
  - `release-notes`: Copies `docs/releases/v<version>.md` to `dist/release/` for upload
  - `release-verify-keys`: Verifies exported public keys contain no secret material
- **Safety scripts**: Added `scripts/verify-gpg-public-key.sh` and `scripts/verify-minisign-public-key.sh`
- **Documentation**: Added Ghostty terminal TERM workaround for macOS signing
